- Open an issue titled "[Feature] Your Suggestion Name."
- Describe the specific CI/CD problem this enhancement solves.

### Your First Code Contribution
1. **Fork** the repo and create your branch from `main`.
2. **Name your branch** following our convention: `feature/your-feature-name`.
3. **Add Tests**: If you are adding a new Github Action, include a sample workflow that uses it.
4. **Linting**: Ensure your YAML is valid (we recommend using `actionlint`).
5. **Submit PR**: Use the provided PR template and link the relevant issue.
